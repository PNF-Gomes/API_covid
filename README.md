# API_covid

Simple APi to suport future API creations and database migrations. 

More instructions will follow, but in order to test please do: 

```
npm init
npm i axios
npm i express

node server.js
```


The routes are the following: 

post:'/covid'

get: '/covid' 
   
get:'/covid/{cityName}'

put: '/notes/{cityName}'

delete: '/notes/{cityName}'
